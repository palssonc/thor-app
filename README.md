# Workout Tracker PWA

A lightweight progressive web app for Intermediate Template B.

## Run locally

```bash
npm run dev
```

Then open [http://localhost:4173](http://localhost:4173).

## Test

```bash
npm test
```

## Install on phone

Serve the folder over your local network or host it anywhere static. Once opened in a supported browser, use the browser install/add-to-home-screen option.

## Publish to GitHub Pages

1. Create a new empty GitHub repository.
2. Push this folder to the repository on the `main` branch.
3. In GitHub, open `Settings -> Pages`.
4. Under `Build and deployment`, choose `GitHub Actions`.
5. The included workflow will publish the app automatically on every push to `main`.

After the first deployment, your app will be available at:

`https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

Install the app from that HTTPS URL on your phone for reliable home-screen and offline support.
